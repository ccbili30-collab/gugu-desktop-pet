========================================
  咕咕桌宠 V2  —  使用说明
========================================


【首次安装】
----------------------------------------
1. 确认电脑已安装 Python 3.10 或更高版本。
   下载地址：https://www.python.org/downloads/
   安装时请勾选 "Add Python to PATH"。

2. 双击运行 install.bat
   - 自动检测 Python 版本
   - 自动安装所需依赖（Pillow、PyYAML）
   - 在桌面创建"咕咕桌宠"快捷方式

3. 安装完成后，打开桌面上的"咕咕桌宠"快捷方式
   进入控制面板，在设置中填写 API Key 后即可开始使用。


【日常使用】
----------------------------------------
双击桌面快捷方式"咕咕桌宠"打开控制面板。

控制面板功能：
  ▶ 启动桌宠   — 让咕咕出现在桌面上
  ■ 停止       — 关闭桌宠
  和咕咕聊天   — 直接在输入框输入，回车或点发送

也可以直接双击宠物本体打开聊天输入框。


【配置 API Key】
----------------------------------------
控制面板 → 设置 → API Key 处填写你的 Key。

支持 DeepSeek、OpenAI 等兼容 OpenAI 接口的服务。

示例（DeepSeek）：
  API Key   : sk-xxxxxxxxxxxxxxxxxxxx
  接口地址  : https://api.deepseek.com
  模型      : deepseek-chat

填写后点"保存设置"，重启桌宠即可生效。

如果不填写 API Key，桌宠仍然可以正常运动，
但不会做出 AI 对话回复。


【修复依赖问题】
----------------------------------------
如果桌宠无法启动，提示缺少模块或依赖，
双击运行 fix_deps.bat，它会：
  - 自动检测缺失的依赖
  - 自动从网络下载并安装
  - 显示安装结果

fix_deps.bat 可以随时单独运行，不影响其他设置。


【热更新（不丢记忆和设置）】
----------------------------------------
收到新版本 zip 包后，不需要重新安装，直接更新：

1. 将新的 gugupet_v2.zip 放到任意位置
   （安装目录、上级目录、下载文件夹或桌面均可）
2. 双击运行 update.bat
3. 脚本自动找到 zip 包并完成更新
   - 代码文件会被新版本覆盖
   - config.yaml、state.json、memory/ 等个人数据不会被清除


【文件说明】
----------------------------------------
install.bat       首次安装，检测环境并创建快捷方式
fix_deps.bat      单独修复缺失依赖，可随时运行
update.bat        热更新到新版本，保留个人数据
start.bat         直接启动桌宠（不经过控制面板）
config.yaml       配置文件（API Key、宠物名等）
pet_control_panel.pyw   控制面板主程序


【常见问题】
----------------------------------------
Q: 双击 install.bat 没有任何反应？
A: 右键 install.bat → 以管理员身份运行。

Q: 提示"python 不是内部命令"？
A: 重新安装 Python，安装时勾选 "Add Python to PATH"。

Q: 桌宠出现了但不回复对话？
A: 检查 config.yaml 中 API Key 是否正确填写，
   以及 llm.enabled 是否为 true。

Q: 桌宠窗口一闪而过或报错退出？
A: 双击 fix_deps.bat 修复依赖，再重新启动。

Q: 如何完全卸载？
A: 删除整个 gugupet_v2 文件夹，
   并手动删除桌面上的"咕咕桌宠"快捷方式即可。
